### 二、简介。

1. 功能
    1. 为接口返回做了mvc分层；
    2. 针对列表、详情等展示做了接口规范（配合前端框架etherMVC使用，可以极大减少前后端对接成本）；
    3. 表单校验规则，按需返回服务端定义的字段规则（rule）用于接口提交数据校验；
    4. 代码生成，可以极大降低服务端开发成本；
    5. 提供几种小部件，配合我们提出的接口规范；
    

### 三、命名空间

```
ethercap\apiBase
```

### 四、安装:

```
composer require ethercap/yii2-api-base:@dev
```
